# 🚗 Real-Time Vehicle Detection & Counting

## 📖 Overview
This project detects, tracks, and counts vehicles in highway traffic using YOLOv8 and computer vision techniques.

## 🎯 Features
- Real-time vehicle detection
- Object tracking
- Vehicle counting
- Bounding box visualization

## 🛠️ Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Supervision

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/AnshulBhaisare/Real-Time-Vehicle-System.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the notebook:
   jupyter notebook

## 📂 Project Structure
- notebooks/ → main notebook
- images/ → output screenshots
- models/ → YOLO weights (not included)
